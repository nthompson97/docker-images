# dev-environment

Basic environment that further projects should use as their base docker image. The main things that are included here,
* A base ubuntu image
* Basic setup including bash history etc (volumes for this might still need to be managed further on)
* The setup for neovim including the basic fork from kickstart, further plugins should be installed for language specific features in projects respectively


See also,
* https://github.com/nvim-lua/kickstart.nvim
